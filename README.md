# 🚀 BugOps AI

> AI-Powered Engineering Operations Platform for intelligent bug triage, release readiness, and engineering decision support.

---

## 📌 Overview

BugOps AI is an AI-assisted Engineering Operations platform that helps engineering teams manage the complete lifecycle of software defects—from bug submission to release approval.

The platform combines intelligent bug analysis, release readiness assessment, engineering insights, and an AI Commander capable of analyzing the entire engineering workspace using real AI models or an offline demo mode.

---

## ✨ Features

### 🐞 Bug Operations
- Create and manage software defects
- Track bug lifecycle
- Severity & priority management
- Ownership assignment
- Status transitions

### 🤖 AI Bug Triage
- AI-assisted bug classification
- Priority recommendations
- Severity analysis
- Engineering reasoning
- Confidence scoring

### 🚀 Release Center
- Release readiness dashboard
- Blocker identification
- Release health monitoring
- Deployment recommendations

### 📊 Engineering Insights
- Engineering health metrics
- Bug distribution
- Risk visualization
- Operational analytics

### 🧠 AI Commander
The executive intelligence engine of BugOps AI.

Capabilities include:

- Workspace-wide engineering analysis
- Executive summaries
- Critical bug identification
- Root cause aggregation
- Release readiness decisions
- Engineering health assessment
- AI-generated recommendations
- Live AI analysis using:
  - OpenAI
  - Grok (xAI)
  - OpenRouter
- Offline Demo Mode

---

## 🎯 AI Commander Modes

### Live AI Mode
Uses your own API key to perform real engineering analysis using large language models.

Supported providers:

- OpenAI
- Grok (xAI)
- Groq (Llama / fast)
- OpenRouter

### Demo Mode

For hackathon demonstrations, AI Commander includes Demo Mode.

Demo Mode:

- requires no API key
- generates realistic executive reports
- demonstrates the complete workflow offline

## 🔐 API Configuration

To enable Live AI analysis:

1. Open **Settings**
2. Navigate to **API Configuration**
3. Select an AI provider
4. Enter your API key
5. Save
6. Open AI Commander and analyze the workspace
